# @diplodoc/lint

Diplodoc platform internal utility set for linting


## Install

```
npm install --save-dev @diplodoc/lint
```

## Usage

Add initial configuration

```sh
npx @diplodoc/lint init
git add --all && git commit -m 'chore: init lint'
```

Run lint

```sh
npm run lint
```

Run lint in fix mode

```sh
npm run lint:fix
```
